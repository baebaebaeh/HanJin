# 운동 영상 플랫폼

## 프로젝트 개요

이 프로젝트는 사용자들이 다양한 운동 영상을 시청하고 리뷰할 수 있는 웹 플랫폼입니다. 사용자 친화적인 인터페이스와 추가 기능을 통해 운동에 관심 있는 사람들에게 유용한 경험을 제공합니다.

## 주요 기능

1. **영상 시청**
   - 다양한 운동 영상 제공
   - 가장 많이 시청된 영상을 메인 페이지에 표시
   - 영상 선택의 편의성 증대

2. **관련 기사**
   - 운동 관련 기사 블록 구현
   - 클릭 시 해당 기사로 직접 이동

3. **사용자 인증**
   - 로그인/로그아웃 기능
   - 회원가입 페이지
   - 로그인과 회원가입 페이지 간 쉬운 전환

4. **리뷰 시스템**
   - 영상 시청 중 리뷰 작성 가능
   - 리뷰 조회, 추가, 수정, 삭제 기능
   - 내용 또는 별점 미입력 시 경고 메시지 표시

## 사용 방법

1. 메인 페이지에서 인기 있는 운동 영상을 확인합니다.
2. 원하는 영상을 선택하여 시청합니다.
3. 영상에 대한 리뷰를 작성할 수 있습니다.
4. 관심 있는 운동 관련 기사를 클릭하여 읽어볼 수 있습니다.
